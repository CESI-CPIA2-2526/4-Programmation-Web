# Bloc 4 - Programmation Web

## Introduction

Ce bloc a pour objectif d'aborder les notions fondamentales du développement Web. D'un point de vue opérationnel, à l'issue de ce bloc, l'étudiant sera en capacité de modéliser et coder une application web, interrogeant une base de données, en respectant les standards de conception.

Les prosits sont contextualisés et permettent d'avancer directement sur certaines pages ou étapes du projet.

Lors des blocs précédents, des séquences sur GIT et la gestion de projet ont été abordées. Le bloc démarre avec un approfondissement de GIT qui se focalise sur le workflow et sur le travail collaboratif. Une nouvelle méthode de travail sera vue avec SCRUM, la plus célèbre des méthodes agiles, qui permettra aux élèves de s'essayer à une autre organisation durant le projet.
